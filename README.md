# FitTrack 🏋️

A fitness tracking web application built with PHP and MySQL.

## Features
- **User Registration & Login** — Secure authentication with hashed passwords
- **Dashboard** — BMI, BMR, daily step goal progress ring, smart insights, weekly chart
- **Activity Logging** — Track steps, calories, distance, and sleep daily
- **Goal Setting** — Set and track daily step goals
- **View History** — Search activity records by date
- **Profile Management** — Edit profile data (name, age, weight, height)
- **Admin Panel** — View, edit, and delete users; view individual user dashboards

## Tech Stack
- **Frontend:** HTML, CSS (Apple Fitness-inspired dark theme), Chart.js
- **Backend:** PHP, MySQL
- **Hosting:** InfinityFree

## Setup
1. Import `setup_database.sql` into phpMyAdmin
2. Update `db.php` with your MySQL credentials
3. Upload all files to your web server
